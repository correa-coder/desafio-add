Documentação da API

Aluno

GET /api/v1/alunos

Retorna uma lista com todos os alunos cadastrados

GET /api/v1/alunos/{id}

Retorna um aluno baseado no ID

POST /api/v1/alunos

Adiciona um aluno ao banco de dados

PUT /api/v1/alunos/{id}

Atualiza dados de um aluno baseado no ID

DELETE /api/v1/alunos/{id}

Remove um aluno baseado no ID

Turma

GET /api/v1/turmas

Retorna uma lista com todas as turmas cadastradas

GET /api/v1/turmas/{id}

Retorna uma turma com base no ID

POST /api/v1/turmas

Adiciona uma turma ao banco de dados

PUT /api/v1/turmas/{id}

Atualiza dados sobre uma turma com base no ID

DELETE /api/v1/turmas/{id}

Remove uma turma com base no ID
